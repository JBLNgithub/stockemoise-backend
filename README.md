backend of the stockemoise website project, more information on my [portfolio](https://jblngithub.github.io/portfolio/)

## Environments variables

required :

```
DATABASE=<path>
PRIVATE_KEY=<key>
```

optional :

```
HOST=<path>
PORT=<port>
NODE_ENV=<ENVIRONMENT>
```
